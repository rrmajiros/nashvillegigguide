# Nashville Gig Guide

The complete Nashville music scene database - show listings, venue guides, studio directory, and industry connections.

## 🚀 Quick Start

### Option 1: Deploy to Vercel (Recommended)
1. Push this repository to GitHub
2. Connect to Vercel at vercel.com
3. Deploy automatically

### Option 2: Local Development
```bash
npm install
npm run dev
```

## 🎯 Features
- **Show Listings**: Comprehensive Nashville concert calendar
- **Venue Guides**: Detailed venue profiles with booking info
- **Studio Directory**: Recording studios and producers
- **Industry Network**: Music professionals directory
- **Artist Discovery**: Emerging Nashville artists

## 📊 Data Sources
- Nashville Scene calendar
- Venue websites
- Radio airplay data
- Industry directories
- User submissions

## 💰 Business Model
- Free basic listings
- Premium venue features

- Industry directory access
- API/data licensing

## 🛠️ Tech Stack
- **Frontend**: Next.js 14 + React + TypeScript
- **Styling**: Tailwind CSS
- **Database**: SQLite (local), PostgreSQL (production)
- **Hosting**: Vercel
- **Email**: Resend + React Email
